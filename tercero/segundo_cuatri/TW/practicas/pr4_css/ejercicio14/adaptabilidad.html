CSS adaptable

Max-With: 750px

Cuando el tamaño de pantalla pase a ser menor de 750 se aplicaran las siguientes características

header img { display: none; }

Ocultará las imágenes que hay en la cabecera

h1 { font-size: 1.5rem; padding: 5px; }

Cambiará el tamaño y el padding de h1, esto también se aplica a h2 y h3 pero con diferentes datoss

main { font-size: 1rem; }

Cambiará el tamaño de la letra del main

div img { max-width: 275px; }

A las imagenes que descienden de un div le cambia la anchura máxima

input, textarea { font-size: 1rem; padding: 5px; }

Reduce el tamaño de los input y textarea

label { padding: 1px; margin: 5px; }

Reduce el padding y margen de los label

label * { padding: 0; margin-left: 5px; }

Reduce el padding y margen izquierdo de los hijos de un label

.aula_libr img { max-width: 200px; }

Reduce el tamaño de las imagenes que descienden de aula_libr

fieldset *{ padding: 2px; }

reduce el padding de los hijos de un fieldset

li a { border: none; color: wheat; }

quita el borde y cambia el color a los enlaces que se encuentran en una lista

footer { font-size: 0.8rem; padding: 5px; }

Reduce el tamaño de la letra y el padding de un pie de pagina

Max-With: 450px

Cuando el tamaño de pantalla pase a ser menor de 450 se aplicaran las siguientes características

h1, h2, h3 { font-size: 1rem; }

Reduce más el tamaño de los encabezados

main { font-size: 0.8rem; }

Reduce más el tamaño de los caracteres del main

div img { max-width: 200px; }

Reduce el tamaño de las imagenes que descienden de un div

.aula_libr img { max-width: 100px; }

Reduce el tamaño de las imagenes que descienden de aula_libr

.aulas_reservadas { font-size: 0.2rem; }

Reduce el tamaño de la fuente de aulas reservadas

td { height: 10px; }

Reduce la altura del td

label * { font-size: 0.8rem; }

Reduce el tamaño de los hijos que descienden de un label

footer { font-size: 0.7rem; }

Reduce el tamaño de los caracteres del pie de pagina
